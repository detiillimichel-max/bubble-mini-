# 🫧 Bubble Shooter Relaxing

Uma versão web do clássico jogo de estourar bolhas, desenvolvida com HTML5, CSS3 e JavaScript puro.  
Perfeito para se divertir rapidamente no navegador.

![Demonstração](screenshot.png) *(adicione uma imagem se quiser)*

## 🎮 Como jogar

- Mova o mouse para **apontar o canhão**.
- **Clique** para atirar a bolha.
- Forme grupos de **3 ou mais bolhas da mesma cor** para eliminá-las.
- Quanto mais bolhas eliminar, mais pontos ganha (10 pontos por bolha).
- Limpe todas as bolhas da tela para **avançar de nível**.
- Use o botão **Next** para ir ao próximo nível (quando todas as bolhas forem eliminadas).
- Botão **Reset** reinicia o jogo do nível 1.

## 🚀 Como executar

1. Baixe o arquivo `index.html`.
2. Abra-o com qualquer navegador moderno (Chrome, Firefox, Edge, etc.).
3. Comece a jogar imediatamente – não precisa de servidor nem instalação.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Canvas API)
